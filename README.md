## ✨ Main features

- **Extreme Performance:** Score of 100 thanks to Astro's static build.
- **Responsive Design:** Adapted to all devices using Tailwind utilities.
- **Clean Code:** Consistent formatting using **Prettier**.

## 🛠️ Technological Stack

- **Core:** [Astro v4+](https://astro.build/)
- **Styles:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons and Mini-videos:** SVGL / Lottiefiles.
- **Mini-videos settings:** TypeScript for better navigation.

## 🚀 Quick start

### 1. Installation of dependencies:

```bash
npm install

```

### 2. Installation of Astro:

```bash
npm create astro@latest

```

### 3. Installation of Tailwind:

```bash
npx astro add tailwind
```
